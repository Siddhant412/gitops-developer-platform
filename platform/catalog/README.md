# Catalog Seed Data

These entities give the Backstage catalog a project-specific starting point.

- `domains/` models the business or platform domain
- `systems/` models higher-level systems
- `groups/` models team ownership
- `users/` models sample identities used by local development
- `resources/` models supporting platform resources such as clusters

`all.yaml` is the root location file loaded by Backstage.
